メッセージ

私は頑張っている人が大好きです！応援したくなります。 日本語を勉強する理由はいろいろあります。「夢のため」「日本語が好き」「日本のアニメが好き」「日本で働きたい」全部素敵です！ 好きな気持ちや前向きな気持ちはどんなにいい教科書よりも役に立ちます。その気持ちを大切にして、いつも楽しく勉強しましょう。 私も皆さんの国の文化が知りたいです。きっとお互いに新しい発見があります。あなたに会えるのを楽しみにしています！
